Perhatian!

Requerement:

1. PHP version min 7.4.xx
2. jQuery version 3.x.x
3. Carbon Fields Plugin

Tested on Worpdress version: 6 above


Intallation:

- Pastikan backup website Anda terlebih dahulu
- Plugin ini membutuhkan plugin Carbon Fields terinstall dan aktif di Worpdress Anda
- Download plugin Carbon Fields (gratis) -> install -> aktifkan plugin Carbon Fields
- Download plugin ini -> install -> aktifkan

Features:
- Custom on load (whatsapp box kondisi open atau close )
- Custom header background color dan text
- Custom background color untuk button
- Bisa paka logo atau tidak pada whatsapp box header
- Custom text header
- Custom enable or disable staff
- Custom bisa upload photo untuk staff atau tidak
- Nama staff
- Job staff
- Pilihan tombol CTA (chat only, call only & chat and call)
